# About: Menu

Firefox extension that adds 'About:' menu into 'Help' menu to access about: pages.

## Compatibility
* Firefox 15.0 - 21.0a2
* SeaMonkey 2.12 - 2.18a2
* Thunderbird 11.0 - 17.*
* Songbird 12.*
