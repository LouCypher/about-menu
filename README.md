# About: Menu

Firefox extension that adds 'About:' menu into 'Help' menu to access about: pages.

## Compatibility
* Firefox 4.0 - 14.0a2
* SeaMonkey 2.1 - 2.9.*
* Thunderbird 9.0 - 12.*
